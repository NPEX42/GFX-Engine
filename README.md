# GFX-Engine

-----

## Installation Instructions (Requires Maven & Git)

### Windows 

```CMD
git clone --recursive https://github.com/NPEX42/GFX-Engine.git
cd GFX-Engine
mvn install
```

```Bash
git clone --recursive https://github.com/NPEX42/GFX-Engine.git
cd GFX-Engine
mvn install
```
